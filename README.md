# Baraqah - FoodPool
An App that brings to life the Food Pool concept developed as Project for Advanced Database Management Systems course.

Concept:
Multiplied blessings through shared meals. A social platform for group dining, authentic reviews, and flexible bill splitting.

## Tech Stack
- **Frontend:** Flutter
- **Backend:** Node.js, Python
- **Databases:** PostgreSQL, MongoDB, Redis

- Phase 2: System Design & Environment Setup (Completed)
